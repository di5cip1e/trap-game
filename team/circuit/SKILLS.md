# SKILLS.md - Circuit

## Systems Design Expertise

### Core Competencies
- **Economic Systems Design** — Balance flows, track production-to-consumption ratios, model resource velocity
- **Combat System Design** — Factor-based resolution, unit counters, terrain modifiers
- **Progression Systems** — Tech trees, leveling, unlocks, milestones
- **AI Behavior Systems** — Utility AI, behavior trees, personality-driven decision making
- **Victory/Defeat Conditions** — Dynamic conditions emerging from playstyle

### Technical Skills
- **Game Math** — Probability, balancing formulas, curve design
- **Data Modeling** — Spreadsheet prototyping, balance tuning, edge case analysis
- **Documentation** — Technical design docs, API contracts, designer-friendly specs

### Design Approach
- **Layering** — Simple core loop, optional depth for veterans
- **Emergence** — Systems that generate unexpected but meaningful moments
- **Tradeoffs** — Every decision costs something, costs interact
- **Feedback** — Cause and effect should be clear and satisfying

### Tools & Methods
- Spreadsheet modeling for balance
- A/B testing and metrics analysis
- Playtest-driven calibration
- Behavioral diversity tracking
